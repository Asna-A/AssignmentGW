Reverse a number

input number
declare reversenumber=0
WHILE number>0
    lastnumber=number MOD 10
    reversenumber=reversenumber*10+lastnumber

Reverse a string

input stringname
declare length
length=length of stringname
FOR each index of stringname
    stringname[i]=stringname[length-i-1]
ENDFOR

Find the given character in the string, return true or false

input stringname,charactername
FOR each index of stringname
    IF stringname[i]==charactername
        RETURN true
    ELSE
        RETURN false

Find the number of occurrences of a character in a given string

input stringname,charactername
Declare count
FOR each index of stringname
    IF stringname[i]==charactername
        count++
RETURN count

Fibonacci series of numbers

Declare a=0,b=1
input numbercount
output a,b
WHILE numbercount>0
    c=a+b
    a=b
    b=c
    output c
    numbercount--
ENDWHILE

Bubble sort
input array
Declare temp
FOR i=0 to length-1
    FOR j=i+1 to length
        IF array[i]>array[j]
            temp=array[i]
            array[i]=array[j]
            array[j]=temp
    ENDFOR
